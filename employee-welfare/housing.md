# 🏡 주거비 지원

#### 지원 대상

| 조건           | 세부 기준                                                                             |
| ------------ | --------------------------------------------------------------------------------- |
| **직급**       | <p><strong>평사원 및 실장, 부실장</strong><br>🛈 지점장은 지원 대상에서 제외</p>                       |
| **거리**       | <p><strong>업장 주소 ↔︎ 실제 거주지</strong><br>네이버지도 ‘반경 거리’ <strong>3 km 이내</strong></p> |
| **세대주 요건**   | 주민등록등본상 **본인이 세대주**인 경우에 한함                                                       |
| **예외**       | 근무지 전보·지점 이동 시 **재검증 후 재승인**                                                      |
| **전세 또는 월세** | **전세 또는 월세 주택에 거주**하는 경우에만 지원                                                     |

***

#### 지원 금액 & 기간

| 항목        | 내용                          |
| --------- | --------------------------- |
| **지원액**   | **월 150,000 원** (세전, 복리후생비) |
| **지급 주기** | 매월 급여일(5일) 함께 입금            |
| **지급 기간** | **무기한** (세대주·거리 요건 충족 시 계속) |

***

#### 신청 절차

1. **신청서 제출**  (#work-업무지원 슬랙 채널에서 주거비 지원 요청 워크플로를 실행해 주세요.)
2. **증빙 첨부**   ① 주민등록등본(최근 1개월) ② 임대차계약서 또는 월세 영수증
3. **HR 검증**  네이버지도 거리 확인 → 3 km 여부 판단
4. **승인 안내**  슬랙 통보, 차월 급여부터 지급

***

#### 유지·변동 규정

| 상황         | 조치                                  |
| ---------- | ----------------------------------- |
| **주소 변경**  | 14일 이내 HR에 신고 → 거리 재검증              |
| **근무지 이동** | 신규 근무지 기준 3 km 재확인 → 요건 미충족 시 지원 중단 |
| **세대주 변경** | 혼인·이사 등으로 세대주가 바뀌면 지원 종료(재신청 가능)    |
| **퇴사·휴직**  | 퇴사 월 급여 지급 시 자동 종료 / 휴직 기간 미지급      |

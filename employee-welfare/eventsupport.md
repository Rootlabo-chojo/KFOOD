# 👨‍👩‍👦 경조사 지원

#### 1️⃣ 지원 내역

| 구분          | 지원 금액·물품           | 경조 휴가                |
| ----------- | ------------------ | -------------------- |
| **결혼(본인)**  | 20 만원 + 화환 (회사 명의) | **3일** (연차 추가 사용 가능) |
| **결혼(자녀)**  | 10 만원 + 화환         | **1일** (연차 추가 사용 가능) |
| **출산**      | 10 만원 + 과일 바구니     | **2일** (연차 추가 사용 가능) |
| **사망**      |  -                 | -                    |
| · 본인·배우자·부모 | 20 만원 + 조화         | **4일**               |
| · 형제자매      | 10 만원              | **3일**               |

***

#### 2️⃣ 신청·지급 절차

1. **경조사 발생** → #work-업무지원 슬랙 채널에서 경조사 지원 요청 워크플로를 실행해 주세요.
2. **첨부파일 제출** (결혼·출산·사망 시)
3. HR 확인 후\
   &#x20;  • 지원금: **급여 통장** 입금\
   &#x20;  • 화환·조화·과일 바구니: 지정 업체 발주
4. **경조 휴가** — 근무 파트너 및 상급자와 일정을 조율한 후, 슬랙 휴가 관리 워크플로에 등록해 주세요.

***

#### 3️⃣ 유의 사항

* 휴가는 **연속 사용 가능**(업장 상황 조율 필수).
* 동일 경조사 **중복·이중 신청 불가**.
* 지급 품목(화환 등)은 **회사 지정 업체**로만 진행.

> 기타 문의 → **#work-업무지원** 채널에 남겨 주세요.

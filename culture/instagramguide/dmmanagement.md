# 📖 DM 관리 방법

#### 1️⃣ DM 수신 영역 구분

Instagram DM은 아래 5가지 영역으로 자동 분류됩니다:

| 구분                        | 설명                    |
| ------------------------- | --------------------- |
| **All (모두)**              | 모든 DM 통합 보기           |
| **Primary (주요)**          | 실시간 알림이 오는 중요 대화      |
| **General (일반)**          | 일반 문의, 비긴급 대응용        |
| **Ads / Promotion (광고)**  | 마케팅으로 유입된 DM          |
| **Message Requests (요청)** | 팔로우하지 않은 사용자로부터의 새 DM |

> 💡 “요청” 영역은 승인하기 전까지 메시지 내용 일부만 보이고, 응답 시 “주요” 또는 “일반”으로 이동할 수 있습니다.

***

#### 2️⃣ 각 영역별 특징과 활용 팁

| 영역               | 특징              | 활용 팁                                     |
| ---------------- | --------------- | ---------------------------------------- |
| **Primary (주요)** | 일반적으로 사용        | 일반적인 유저 대화                               |
| **General (일반)** | 특징 있는 유저만       | 일반적으로 활용하지 않습니다.특징 있는 유저만 일반으로 이동        |
| **Ads / 광고**     | 마케팅으로 유입 된 유저   | 대화방이 생기면 먼저 DM 메시지를 보냅니다.                |
| **요청**           | 팔로우 없이 수신된 새 DM | <p>승인 후 주요탭으로 이동<br>(특별한 경우 일반으로 이동)</p> |

> ⚠️ **주요 탭을 제외한 모든 영역은 DM이 와도 알림이 오지 않습니다.**\
> 따라서 **매번 직접 들어가서 체크해야** 메시지를 확인할 수 있습니다.

***

#### 3️⃣ DM 확인 루틴

**효율적인 대응을 위한 5단계 루틴입니다.**

1. ✅ **All(모두)** 탭에서 전체 흐름 확인
2. 📬 **Primary(주요)** 에 있는 메시지부터 확인
3. 📥 **General(일반)** 탭에서 순차적 대응
4. 📣 **Ads / 광고(광고)** 상대방이 메시지를 보내지 않아도 먼저 DM 메시지를 보냅니다.
5. 🔍 **요청** 메시지 → 수락 여부 결정 후 분류 이동(주요 탭 이동)

> 📌 추천 확인 타이밍: 오전 업무 전 / 점심 직후 / 퇴근 직전

***

#### 4️⃣ 메시지 분류 및 이동 방법

* 📱 **모바일**: 메시지 좌/우 스와이프 → '이동' 또는 '주요 표시'
* 💻 **PC/웹**: 채팅 내 상단 메뉴에서 ‘Move to Primary/General’ 선택



***

#### ✅ 실전 운영 팁 요약

* "요청 및 주요"는 실시간 확인 필수, 나머지는 정기 루틴으로 점검
* 메시지 처음 받을 때 **즉시 분류**하면 추후 누락 방지
* DM도 브랜드 응대의 연장선 → **응대 톤 & 반응속도 관리 중요**

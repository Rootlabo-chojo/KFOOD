# 📖 인스타그램 가이드

* **주요 내용:**
  * 브랜드 톤앤매너 및 콘텐츠 가이드라인
  * 포스팅, 스토리, 릴스 제작 표준 절차
  * 필수 해시태그 및 이미지 사용 규정
  * 댓글 및 DM 응대 매뉴얼

**이 가이드는 살아있는 문서입니다. 언제든지 더 좋은 의견이 있다면 자유롭게 제안해주세요!**

* 왼쪽 메뉴 탐색 : 좌측의 카테고리를 클릭하여 필요한 정보를 주제별로 찾아볼 수 있습니다.
* 검색 기능 활용 : 상단의 `검색(Search)` 기능을 이용하면 원하는 키워드가 포함된 문서를 빠르게 찾을 수 있습니다.
* 피드백 및 수정 제안 : 내용에 수정이 필요하거나 추가하고 싶은 정보가 있다면, 언제든지 담당자에게 알려주세요. 함께 더 나은 가이드를 만들어가요!

> **💡 Tip: 문서는 계속 업데이트됩니다!**
>
> 회사는 계속 성장하고, 우리의 일하는 방식도 더 나은 방향으로 발전합니다. 이 가이드 역시 정기적으로 업데이트되니, 즐겨찾기 해두고 자주 확인해주세요.

> **🚀 목표: 단순한 '좋아요'가 아닌 '팬'을 만드는 것**
>
> 모든 콘텐츠는 우리 브랜드를 더 깊이 이해하고 사랑하게 만드는 것을 목표로 합니다.

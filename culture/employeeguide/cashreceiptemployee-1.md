# 📖 Copy of 현금영수증 발행 방법

#### 현금영수증 발행 흐름

| 단계         | 해야 할 일                                                               | 화면 위치       |
| ---------- | -------------------------------------------------------------------- | ----------- |
| 결제내역 메뉴 진입 | PayHere 좌측 메뉴 → **`결제내역`**                                           | 대시보드        |
| 기간 설정      | 좌측 상단 **`1주`** 버튼 클릭 → 달력에서 **`2024-04-02`** 선택                      | 필터 바        |
| 거래 찾기      | 리스트에서 거래 찾기                                                          | 결제 리스트      |
| 영수증 보기     | 행 우측 **`영수증 보기`** 클릭                                                 | 팝업 또는 새 창   |
| 현금영수증 발급   | 영수증 화면 우측 **`현금영수증 발급`** 버튼 클릭                                       | 영수증 팝업      |
| 정보 입력      | <ul><li>지출증빙 옵션이 보이면 선택</li><li>사업자 등록번호 or 휴대폰 번호 입력 → 발급</li></ul> | 발급 창        |
| 완료 확인      | “발급 완료” 토스트 메시지 확인 → 고객·회계팀에게 공유                                     | 상단 또는 하단 알림 |

{% hint style="info" %}
**문의** 발급이 안 되거나 화면이 다를 경우 **즉시** <@본사-조병조-CEO>에게 알려 주세요.
{% endhint %}

# 📖 네이버 쇼핑 사용 방법

#### **1️⃣** 결제 전 체크리스트

| 항목     | 점검 질문                        | 확인 방법           |
| ------ | ---------------------------- | --------------- |
| 업장 구분  | 내가 결제할 상품은 어느 지점/브랜드인가?      |                 |
| 카드 유효성 | 해당 매장 전용 카드가 네이버페이에 등록돼 있는가? | 네이버페이 ▸ 결제수단 관리 |
| 재고·옵션  | 옵션(수량/맛 등)이 정확한가?            | 주문서 미리보기        |

***

#### 2️⃣ 네이버페이 결제 절차

1. **상품 이동**
   * 링크 클릭 또는 네이버쇼핑에서 상품 검색 → **`구매하기`**
2. **배송지 선택**
   * 기본 배송지 확인 → 필요 시 `주소록`에서 변경
3. **결제수단 선택**
   * **`네이버페이 간편결제`** → 카드 탭
4.  **업장 전용 카드 선택**

    * 지점명 / 마스킹 번호를 확인하고 선택

    <figure><img src="/broken/files/qtAA1hbux5UKXkbBQFz2" alt=""><figcaption></figcaption></figure>
5. **할부, 포인트 설정**
   * 할부는 허용 하지  않습니다.(회계 지침)
   * 포인트 사용 금지 (정산 혼선 방지)
6. **결제 완료**

***

#### 3️⃣ 카드 선택 가이드

| 상황         | 사용 카드                        | 비고            |
| ---------- | ---------------------------- | ------------- |
| 매장 재료, 소모품 | 매장 전용 법인 체크카드                | 카드 별칭에 지점명 표시 |
| 본사 공통 소모품  | 본사 공용 카드(케이에프오오디)            | 지점명 없음        |
| 긴급 개인 결제   | 개인 카드 결제 → #billing-개인청구에 요청 | 사전에사유유 보고     |

> **Why?**\
> 업장별 카드 사용을 지키면 네이버 정산 리포트가 지점별로 자동 분리되어, **수기 매칭 작업**이 사라집니다.

***

#### 5️⃣ 유의사항 ⚠️

1. 할부 결제를 지원하지 않습니다.
2. 업장별 카드 선택은 정산 지연, 오류를 예방하는 최우선 수칙입니다.
3. 결제 실패가 반복될 경우 **즉시** <@본사-조병조-CEO>에게 알려 주세요.

# 📖 아사나(Asana) 기본 원칙

1️⃣ 아사나 활용 3대 핵심 원칙

우리 팀은 아래 3가지 원칙을 기반으로 아사나를 활용합니다.

| 원칙        | 설명                                                  |
| --------- | --------------------------------------------------- |
| 100% 투명성  | 모든 업무는 아사나에 테스크로 생성하여 모든 팀원이 현황을 알 수 있도록 합니다.       |
| 명확한 책임    | 모든 테스크에는 반드시 담당자(1명)와 마감일을 지정하여 책임 소재를 분명히 합니다.     |
| 실시간 현황 공유 | 업무 상태가 변경될 때마다 칸반보드에서 테스크를 이동시켜 별도 보고 없이 현황을 공유합니다. |

> 👍 아사나 활용 목표: "그거 어떻게 되고 있어요?" 라는 질문이 사라지는 것.

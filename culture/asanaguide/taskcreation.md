# 📖 테스크(Task) 생성 규칙

1️⃣ 테스크 작성 필수 항목

테스크 생성 시 아래 5가지 항목은 반드시 포함되어야 합니다.

| 항목     | 설명                                                | 예시                          |
| ------ | ------------------------------------------------- | --------------------------- |
| 테스크 제목 | 누가 봐도 알 수 있도록 명확하고 구체적으로 작성합니다.                   | `[블로그] 8월 신메뉴 소개 콘텐츠 초안 작성` |
| 담당자    | 해당 업무를 책임지고 완수할 담당자 1명을 지정합니다.                    | `김대업`                       |
| 마감일    | 업무가 완료되어야 하는 날짜를 명확히 설정합니다.                       | `2025-08-29`                |
| 프로젝트   | 해당 테스크가 속한 프로젝트를 연결합니다.                           | `콘텐츠 마케팅`                   |
| 설명     | 업무의 배경, 목표, 참고 자료(링크, 파일) 등 상세 내용을 최대한 자세히 기재합니다. | `지난 회의록 참고하여...`            |

> ⚠️ 주의: 유령 테스크 금지
>
> 담당자나 마감일이 지정되지 않은 '유령 테스크'는 업무 누락의 원인이 됩니다. 반드시 모든 항목을 채워주세요.

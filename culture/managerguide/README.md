# 📖 관리자 가이드

* **주요 내용**
  * POS 시스템 운영 및 매출 관리
  * 직원 계정 생성 및 권한 설정
  * 재고 및 발주 시스템 관리
  * 프로모션 및 이벤트 설정 방법

**이 가이드는 살아있는 문서입니다. 언제든지 더 좋은 의견이 있다면 자유롭게 제안해주세요!**

* 피드백 및 수정 제안 : 내용에 수정이 필요하거나 추가하고 싶은 정보가 있다면, 언제든지 담당자에게 알려주세요. 함께 더 나은 가이드를 만들어가요!

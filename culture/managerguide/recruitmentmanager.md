# 📖 채용 공고·채용 방법

#### 1️⃣ 목적

* **통합 가이드** : 채용 공고 작성부터 게시 · 관리까지 일관된 프로세스 제공
* **업무 효율화** : 불필요한 문의·반복 작업 최소화
* **브랜드 일관성** : 모든 플랫폼에 동일한 톤-앤-매너, 정보 구조 유지

***

#### 2️⃣ 역할·책임

| 구분        | 담당                               | 책임         |
| --------- | -------------------------------- | ---------- |
| **개별 업장** | 채용 공고 작성·검수, 전 플랫폼 게시·수정, 성과 리포트 | 본사-신주혁-부서장 |
| **채용 요청** | JD 초안 작성 → 채용공고 게시               | 본사-신주혁-부서장 |

#### 3️⃣ 채용 공고 작성 절차

1. **JD(Job Description) 초안 작성**
   * 요청 업장는 아래 JD 양식 참조
   * 필수 항목: 직무명·주요 업무·자격요건·우대사항·근무형태·근무지·연봉 범위
2. **채용 지원(선택)**
   * 직무 특성상 세부 설명이 필요한 경우 → 본사 지원 요청
3. **JD 브랜드 일관 성 유지**
   * 브랜드 톤(반말/존칭, 아이콘 등)·채용 정책(채용 절차, 복지) 항목 일괄 삽입
4. **사내 공용 계정으로 채용공고 게시**
   * 플랫폼별 포맷 차이에 맞춰 복사·붙여넣기 후 미리보기 검수
5. **게시 확인 & 링크 공유**
   * Slack `#hr-인사-지원` 채널에 공고 링크 + 게시 일시 공유
6. **채용 확정 시 사원 등록 및 Slack 인사 등록**
   * [사원 등록](https://v3n4mwznbhr.typeform.com/to/sbYzLbeV)을 진행하고, Slack의 `#hr-인사-공지` 채널에 채용 공고를 공유해 주세요.

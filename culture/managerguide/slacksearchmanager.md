# 📖 슬랙(Slack) 검색 기능 활용법

#### 1️⃣ 왜 슬랙 검색을 마스터해야 할까요?

우리 회사는 "질문하기 전에 검색하기 (Search before you ask)" 문화를 지향합니다. 누군가에게 질문하고 답을 기다리는 시간, 그리고 질문에 답하느라 다른 동료의 업무 흐름을 끊는 시간을 절약할 수 있기 때문입니다.

* **업무 효율화 :** 내가 원하는 정보, 파일, 결정 사항을 즉시 찾아내어 업무의 속도를 높입니다.
* **정보 투명성 :** 과거의 모든 논의와 결정 이력을 투명하게 확인함으로써, 히스토리를 파악하고 더 나은 의사결정을 할 수 있습니다.
* **협업 존중 :** 동료의 집중 시간을 존중하고, 스스로 문제를 해결하는 문화를 만들어갑니다.

***

#### 2️⃣ 슬랙 검색의 기본기 : 필터 활용하기

가장 기본적인 검색 방법은 키워드를 입력한 후, 슬랙이 제공하는 필터를 사용하는 것입니다.

1. 슬랙 검색창(단축키: `Ctrl` + `G` 또는 `Cmd` + `G`)에 찾고 싶은 키워드(예: `3분기 보고서`)를 입력합니다.
2. 검색 결과 상단에 나타나는 \[메시지], \[파일], \[채널], \[사람] 탭을 눌러 원하는 결과만 필터링합니다.
3. 우측의 \[필터] 패널을 열어 특정 채널, 특정 기간 등 더 상세한 조건으로 결과를 좁힐 수 있습니다.

(이미지 예시: 검색 결과 상단의 필터 탭)

***

#### 3️⃣ 핵심 기능: 검색 연산자 (Search Modifiers)

진정한 슬랙 고수는 '검색 연산자'를 사용합니다. 키워드와 연산자를 조합하면 놀랍도록 정확한 검색이 가능합니다.

| 연산자 (Modifier) | 기능                    | 사용 예시                                       |
| -------------- | --------------------- | ------------------------------------------- |
| `in:`          | 특정 채널이나 특정인과의 대화에서 검색 | `in:#마케팅-채널` 또는 `in:@오세기`                   |
| `from:`        | 특정인이 보낸 메시지만 검색       | `from:@조병조`                                 |
| `to:`          | 나에게 온 멘션(@)이나 DM 검색   | `to:me`                                     |
| `has:`         | 특정 요소를 포함한 메시지 검색     | `has:link` (링크), `has:📌` (핀), `has:⭐` (저장) |
| `before:`      | 특정 날짜 이전의 메시지 검색      | `before:2025-07-15`                         |
| `after:`       | 특정 날짜 이후의 메시지 검색      | `after:2025-06-01`                          |
| `on:`          | 특정 날짜에 작성된 메시지 검색     | `on:yesterday` 또는 `on:2025-07-01`           |
| `" "`          | 반드시 포함된 정확한 문구를 검색    | `"3분기 최종 보고서"`                              |
| `-`            | 특정 단어를 제외하고 검색        | `보고서 -1분기` (1분기를 제외한 보고서 검색)                |

***

#### 4️⃣ 실전 활용 예시: 연산자 조합하기

연산자를 조합하면 마법 같은 검색이 가능해집니다.

*   CASE 1: 김대표님이 #전략기획 채널에 올린 '법인카드' 관련 링크를 찾고 싶을 때

    > `in:#전략기획 from:@김대표 has:link 법인카드`
*   CASE 2: 지난 6월 한 달 동안, 오세기님과 DM으로 나눈 '서버 비용' 관련 대화를 찾고 싶을 때

    > `in:@오세기 서버 비용 after:2025-05-31 before:2025-07-01`
*   CASE 3: "프로젝트 블루 최종 제안서" 라는 이름의 파일을 찾고 싶을 때

    > `"프로젝트 블루 최종 제안서" has:file`
*   CASE 4: 어제, 내가 핀(pin)해두었던 모든 메시지를 보고 싶을 때

    > `on:yesterday has:pin`

***

#### 5️⃣ 알아두면 유용한 추가 팁

* 검색 결과 정렬: 검색 결과 우측 상단의 정렬 기준을 `가장 관련성 높은` 또는 `가장 최근`으로 변경하여 볼 수 있습니다.
* 파일만 따로 검색: 슬랙 좌측 상단의 `더보기` > `파일` 로 이동하면 파일에 특화된 검색 및 필터링이 가능합니다.
* 나중에 검색하기: 중요한 메시지는 `나중에 하도록 저장(별 모양 아이콘)` 해두면, 나중에 `저장된 항목`에서 쉽게 찾아볼 수 있습니다. 이것도 일종의 검색입니다.



이 가이드를 바탕으로 우리 모두 '슬랙 검색 마스터'가 되어, 정보 검색에 쓰는 시간을 줄이고, 더 중요한 가치를 만드는 데 집중하기를 바랍니다!

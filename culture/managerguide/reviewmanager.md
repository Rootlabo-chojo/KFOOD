# 📖 리뷰 관리 방법

#### 1️⃣ 일일 리뷰 체크리스트

| 시간대     | 해야 할 일                                                                      | 실무 팁                                                                                           |
| ------- | --------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| 영업 시작 전 | <p>1. 각 플랫폼 리뷰 확인 (리뷰 트래킹)<br>2. 팀 회의를 통한 리뷰 내용 공유<br>3. 관리자 계정으로 답변 작성</p> | <ul><li>별점 3↓ 리뷰 발생 시 리포트에 상세 기재합니다. (공유 필수)</li><li>긍정 리뷰는 감사 멘트 + 강조 포인트(맛, 서비스 등)</li></ul> |
| 마감 보고   | <p>일일 리포트(슬랙) 작성 시<br><code>리뷰 확인 완료</code> 체크</p>                          | 확인 없이 진행하거나 체크하지 않도록 해주시기 바랍니다.                                                                |

***

#### 2️⃣ 리뷰 플랫폼 목록 & 접속 경로

| 플랫폼                                          | 접속 / 앱                                                | 계정·권한         | 비고                         |
| -------------------------------------------- | ----------------------------------------------------- | ------------- | -------------------------- |
| <p><strong>배달 플랫폼</strong><br>(배민·요기요 등)</p> | 사장님 앱 → 리뷰관리                                          | 매장별 관리자       | 모든 배달 플랫폼 구매 후기 모두 확인 및 답변 |
| **네이버**                                      | SmartPlace → 리뷰                                       | 회사 통합 네이버 ID  | 방문 리뷰 / 블로그 리뷰 모두 확인 및 답변  |
| **캐치테이블**                                    | [biz.catchtable.co.kr](https://manager.tablenote.co/) | 매장 관리자        | 예약 리뷰 모두 확인 및 답변           |
| **기타(구글 등)**                                 | 각 관리자 콘솔                                              | 본사 요청 시 권한 제공 | 방문 리뷰 답변 모두 확인 및 답변        |

**Tip** 리뷰 페이지를 모바일 홈 화면에 ‘바로가기’로 추가해 두면 출근길에도빠르게 확인할 수 있습니다.

***

#### **3️⃣** 답변 작성 가이드

| 상황                       | 톤 & 매너            | 예시 템플릿                                                             |
| ------------------------ | ----------------- | ------------------------------------------------------------------ |
| 긍정 리뷰                    | 감사 + 인상적인 포인트 재확인 | “소중한 후기 감사드립니다! ☺️ 말씀해주신 ✦{칭찬 포인트}✦ 를 꾸준히 유지하겠습니다.”                |
| 중립 리뷰                    | 개선 의지 + 추가 질문     | “불편을 드려 죄송합니다. 🙏 더 나은 경험을 위해 구체적인 상황을 알려주시면 즉시 개선하겠습니다.”          |
| <p>부정 리뷰<br>(★ 3 이하)</p> | 사과 + 해결 제안 + 연락처  | “죄송합니다. 해당 문제는 내부 확인 후 바로 조치하였습니다. 추가 문의는 000-000-0000 으로 연락 주세요.” |

***

#### 4️⃣ 유의사항 ⚠️

* 답변은 \_관리자 권한\_이 있는 계정만 작성 가능합니다.
* 플랫폼 초대장은 슬랙 등록 연락처로 발송되며, 수락 후 바로 리뷰 관리를 시작해 주세요.
* 고객 개인정보(전화·주소 등)는 답글에 노출하지 않습니다.

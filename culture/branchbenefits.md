#### 업무 지원 & 복지

| 제도           | 혜택                              |
| ------------ | ------------------------------- |
| **업무 지원 제도** | 본사 통합 시스템·법인카드·마케팅 툴 등 전 이용 가능  |
| **연차 & 휴무**  | 주 5일 근무, 정규 휴무 + 연차 사용 → 근무 유연성 |
| **채용 지원**    | 내부 ATS·협력 플랫폼을 통한 채용 공고·필터링 지원  |
| **마케팅 지원**   | 본사에서 채널 세팅·광고 운영케어              |

#### 투자 금액별 인센티브

| 투자 비율 (총 1.5 \~ 2 억 기준) | 예상 투자액                 | 인센티브     |
| ----------------------- | ---------------------- | -------- |
| **33 %**                | 약 5,000 만 \~ 6,600 만 원 | **25 %** |
| **66 %**                | 약 1 \~ 1.3 억           | **50 %** |
| **90 %**                | 약 1.35 \~ 1.8 억        | **80 %** |

> 투자 비율에 따라 **지분·수익 배분** 및 운영권 범위가 조정되며, 세부 조건은 경영진 협의를 통해 확정됩니다.

***

#### 개인사업자 vs. 직영점 비교

| 개인사업자 어려움         | 직영점 해결 포인트                    |
| ----------------- | ----------------------------- |
| 신규 법인·시장 분석 직접 진행 | **컨설팅 무료 제공** (시장·입지·인허가)     |
| 메뉴 개발·식자재 관리 부담   | 본사 **R\&D·공동구매** 시스템 지원       |
| 인테리어 감리·브랜딩 부재    | **표준 BI·SI** + 본사 감리팀 투입      |
| 전문 마케팅 역량 부족      | **마케팅회사 대표가 직접** 광고·SNS 운영 케어 |
| 채용 어려움            | **본사 HR** 풀 및 브랜드 네임 활용       |

> 🫂 **결론**: 직영점 지점장은 초기 투자와 리스크를 대폭 줄이고, 본사의 전문 지원으로 **안정적인 수익과 탄탄한 성장**을 동시에 확보할 수 있습니다.

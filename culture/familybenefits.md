# 🥳 가족 및 지인 방문 혜택

#### 예약 / 웨이팅 혜택

| 단계           | 가이드라인                                                                                                                                       |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| **사전 연락**    | 방문 전 **업장 관리자(부실장 이상)**&#xC5D0;게 예약·웨이팅 가능 여부를 문의                                                                                           |
| **예약 지원**    | <p>예약 가능한 시간·테이블이 있을 때만 지원<br>※ 기존 고객의 예약을 취소·변경하여 자리 배정 ❌</p>                                                                              |
| **웨이팅 우선권**  | <p>- <strong>가족</strong>: 1순위 우선 입장 허용<br>- <strong>지인</strong>: 비정기적 · 한시적 우선권 가능 (동일 인물 반복 불가)<br>→ 최종 판단은 <strong>업장 관리자</strong> 결정</p> |
| **방문 태도 안내** | 혜택 안내 시 “정상 예약/웨이팅처럼 자연스럽게 방문” 요청 필수 → 업장 분위기 유지                                                                                            |

***

#### 서비스(음식·주류) 혜택

| 구분            | 운영 원칙                                                        |
| ------------- | ------------------------------------------------------------ |
| **판단 주체**     | **키친팀·홀팀**이 당일 업무량·재고 상황을 보고 자율 결정                           |
| **우선순위**      | 일반 고객 서비스가 **항상 1순위** — 가족·지인 서비스는 후 순위                      |
| **제공 예시**     | <p>- 음식 3개 주문 → 메뉴 1개 서비스<br>- 주류 1병 주문 → 추가 잔/글래스 1회 제공</p> |
| **과다 서비스 금지** | 테스트 메뉴·재고 소진용 등 **합리적인 범위**에서만 제공                            |
| **커뮤니케이션**    | 제공 시 반드시 **가족/지인 취향 파악 → 과·소 서비스 방지**                        |

***

#### 운영 원칙 요약

1. **업장 손해가 없는 선에서 구성원 사기·애사심을 높이는 목적**
2. **가족・지인 혜택 = 예외**, 반복·과도 적용 금지
3. 관리자(부실장 이상) 판단·승인을 거쳐 일관성 있게 운영
4. 모든 직원은 동일 기준을 적용하여 **형평성** 유지

> 문의 또는 예외 상황 발생 시 ➜ **대표·업장 관리자에게 즉시 공유**하고 지침에 따라 주세요.

# 📖 명함 신청 방법

#### 1️⃣ 명함 신청 경로

* 명함 신청은 Slack 채널 **`#work-업무지원`** 에서 진행합니다.
* 해당 채널 내 **업무지원 요청 워크플로우(Workflow)** 를 실행합니다.

***

#### 2️⃣ 워크플로우 선택

* 워크플로우 실행 시 **요청 유형(Type)** 에서 → **“명함 요청”** 을 선택합니다.

***

#### 3️⃣ 상세 내용 입력

**상세 내용란에는 아래 정보를 반드시 기입해주세요.**

| 항목        | 예시                   |
| --------- | -------------------- |
| 이름(국문/영문) | 조병조 / Byungjo Cho    |
| 소속        | 올리버바이쇼텐  압구정         |
| 직급        | 대표 / 실장 / 매니저 등      |
| 연락처       | 010-1234-5678        |
| 이메일       | chojo@kfoodinc.co.kr |

💡 _입력 예시_

> 이름: 조병조 (Byungjo Cho)\
> 소속: 올리버바이쇼텐압구정\
> 직급: 대표\
> 연락처: 010-1234-5678\
> 이메일: chojo@ kfoodinc.co.kr

***

#### 4️⃣ 제작 및 전달

* 운영팀에서 요청 내용을 확인 후, 디자인팀을 통해 명함이 제작됩니다.
* 제작 완료 시 Slack DM 또는 비대면 수령(택배/내부 전달)을 통해 전달드립니다.

***

#### 5️⃣ 기타 안내

* 명함 디자인은 **통합 브랜드 CI 기준**으로 통일됩니다.
* 명함 정보 수정(직급 변경, 연락처 변경 등)은 동일한 절차로 재신청 가능합니다.
* 제작 소요 기간은 **영업일 기준 약 7일\~**&#xC785;니다.

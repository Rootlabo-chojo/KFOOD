# 📖 매장 사진 촬영 및 AI 보정 매뉴얼

> **목적**: 매장별 사진 품질 편차 제거 + 채널 간 비주얼 통일성 확보

***

#### 1️⃣ 사진 종류 결정

촬영 시작 전, 어떤 용도의 사진인지 먼저 결정합니다.

| 구분          | 네이버 메뉴판용            | 브랜드/마케팅용            |
| ----------- | ------------------- | ------------------- |
| **목적**      | 정보 전달               | 감성·전환               |
| **앵글**      | 탑뷰 (90°)            | 45° 측면              |
| **강조 포인트**  | 메뉴 구성 한눈에           | 음식 질감·분위기           |
| **사용처**     | 네이버 메뉴판, 페이히어 메뉴 등록 | 인스타그램, 배민 메인 배너, 광고 |
| **AI 보정 톤** | 깨끗하고 명료한 톤          | 따뜻하고 분위기 있는 톤       |

***

#### 2️⃣ 촬영 가이드 — 네이버 메뉴판용 (탑뷰)

**촬영 전 준비**

* [ ] 카메라 렌즈 한번 닦기 (지문·먼지 제거)
* [ ] 음식 주변 지저분한 요소 정리
  * 휴지, 컵, 행주, 양념통, 영수증 등 제거
* [ ] 접시 가장자리 소스 자국 닦기
* [ ] 음식 플레이팅 최종 확인

**카메라 설정**

| 항목       | 설정               |
| -------- | ---------------- |
| 카메라 앱    | 기본 카메라 사용        |
| 필터       | 사용 금지            |
| 줌(확대/축소) | 사용 금지            |
| 플래시      | 사용 금지            |
| 화질       | 최고 화질 (원본 비율 유지) |

**촬영 방법**

1. 음식 **바로 위**에서 촬영 (탑뷰, 수직 90°)
2. 음식이 화면 **정중앙**에 오도록 구도 잡기
3. 접시가 화면에 **꽉 차게** 촬영
4. 흔들림 방지 — 두 손으로 안정적으로 잡거나 테이블에 팔꿈치 고정

**조명**

* 매장 기본 조명 그대로 사용 (추가 조명 X)
* 너무 어두우면 음식 위치를 조금 옮겨 매장 조명 아래로 이동
* 손 그림자가 음식 위에 떨어지지 않도록 주의

***

#### 3️⃣ 촬영 가이드 — 브랜드/마케팅용 (45도)

**촬영 전 준비**

* [ ] 카메라 렌즈 한번 닦기
* [ ] 음식 주변 지저분한 요소 완전 정리
* [ ] 배경 정돈 (행주, 잡동사니 등 프레임 밖으로)
* [ ] 음식의 가장 보기 좋은 면 확인

**카메라 설정**

| 항목          | 설정                |
| ----------- | ----------------- |
| 카메라 앱       | 기본 카메라 사용         |
| 필터          | 사용 금지             |
| 줌           | 사용 금지             |
| 플래시         | 사용 금지             |
| 인물 모드/아웃포커싱 | 사용 권장 (스마트폰 지원 시) |

**촬영 방법**

1. 음식과 **45도 각도**에서 촬영
2. 음식의 질감·입체감이 잘 보이는 면을 정면으로
3. 프레임의 약 60\~70%를 음식이 차지하도록
4. 배경에 매장 분위기가 살짝 보이도록 여백 확보

**조명**

* 매장 자연광이 있는 자리 선호 (창가 등)
* 조명이 음식 측면에서 들어오면 입체감 강조
* 정면 직사광은 피하기

***

#### 4️⃣ AI 보정 워크플로우 (LifeLENS Playground)

**접속**

**URL**: `https://api.lifelens.co.kr/playground`

**STEP 1. 사진 업로드 준비**

* 촬영한 원본 사진을 **그대로** 준비 (별도 편집 금지)
* 이미지가 깨지지 않도록 원본 파일 사용

**STEP 2. 기준 결과물 선택**

* AI Playground 접속 후 **'최근 결과'** 영역 확인
* 통일성 있게 구성된 메뉴 사진 결과물 찾기
* 해당 결과물에서 **'폼에 그대로 채우기'** 버튼 클릭

**STEP 3. 이미지 교체**

* 폼에 자동으로 배경 이미지 + 사진 이미지 두 개가 채워짐
* **사진 이미지 옆 'X' 버튼 클릭하여 제거**
* 비워진 자리에 **방금 촬영한 원본 사진 업로드**
* 배경 이미지는 그대로 유지

**STEP 4. 요청 전송**

* **'요청 보내기'** 버튼 클릭
* 결과 생성 대기 (수 초 \~ 1분 내외)

**STEP 5. 결과 다운로드**

* 생성된 이미지 확인
* **다운로드** 버튼 클릭하여 저장

**재보정이 필요한 경우**

결과가 만족스럽지 않으면:

1. 다른 기준 결과물 선택 후 STEP 2부터 재시작
2. 또는 원본 사진을 다시 촬영하여 처음부터 진행
3. 3회 이상 실패 시 @본사-조병조-CEO 호출

#### 5️⃣ 최종 사용처별 가이드

**네이버 플레이스 메뉴판**

* 사용 사진: **탑뷰 보정본**
* 업로드: 네이버 플레이스 비즈니스 센터 → 메뉴 관리
* 권장 크기: 1080 x 1080 이상

**페이히어(토스플레이스)**

* 사용 사진: **탑뷰 보정본**
* 업로드: 페이히어 셀러 센터 → 상품 이미지
* 메뉴별 1장씩 통일된 톤 유지

**배달의민족**

* 메뉴 사진: **탑뷰 보정본**
* 메인 배너: **45도 보정본**
* 업로드: 배민사장님광장 → 매장 정보

#### 6️⃣ 자주 발생하는 실수 (FAQ)

**Q1. 촬영본이 너무 어둡게 나옵니다**

매장 조명 아래로 음식 위치 이동. 플래시는 절대 사용 금지.

**Q2. AI 보정 결과가 원본과 너무 다르게 나옵니다**

'최근 결과'에서 다른 기준 결과물 선택. 톤이 가까운 결과물 우선.

**Q3. 배경 이미지를 바꿔도 되나요?**

통일성 유지를 위해 같은 카테고리 메뉴는 동일 배경 사용 권장.

**Q4. 보정 전 사진을 포토샵으로 편집해도 되나요?**

절대 금지. AI에 원본 그대로 입력해야 결과 일관성 확보.

**Q5. 셀카 모드(전면 카메라)로 찍어도 되나요?**

후면 카메라 사용. 전면 카메라 화질 부족.

#### 7️⃣ 최종 체크리스트

촬영 → 보정 → 업로드 전 마지막 점검:

* [ ] 원본 사진이 흔들리지 않았는가
* [ ] 음식 외 불필요한 요소가 프레임에 없는가
* [ ] AI 보정 결과 톤이 다른 메뉴 사진과 통일되는가
* [ ] 파일명 규칙(매장명\_메뉴명\_날짜)을 지켰는가
* [ ] 올바른 채널에 올바른 앵글(탑뷰/45도)로 업로드했는가

# ☕ 커피챗 요청

#### 커피챗 요청 기준

| 구분         | 예시 주제             | 비고         |
| ---------- | ----------------- | ---------- |
| **업무 개선**  | 프로세스 병목, 시스템 제안   | —          |
| **커리어 상담** | 직무 방향, 성과·평가 질문   | —          |
| **조직 문화**  | 팀 분위기, 갈등 완화 아이디어 | —          |
| **개인 고충**  | 워라밸, 건강, 관계 스트레스  | 필요 시 HR 연결 |

> **요청 시 반드시 “왜 만나고 싶은지”를 1–2 줄로 구체화**해 주세요.\
> (예: “매장 재고 관리 POS 개선 아이디어 공유 건”, “다음 분기 목표 설정 상담” 등)

***

#### 워크플로 옵션

| 워크플로       | 요청 가능 대상   | 메시지 노출         | 사용 상황               |
| ---------- | ---------- | -------------- | ------------------- |
| **공개 요청**  | **관리자 전체** | #관리자 슬랙 채널에 공개 | 아이디어 공유, 전사 협업 주제   |
| **비공개 요청** | **경영진 전용** | #경영진 슬랙 채널에 공개 | 민감 이슈, 개인 고충, 인사 관련 |

* 슬랙 커맨드로 호출하면 **자동 템플릿**이 뜹니다 → 목적·희망 일정 입력 → 담당자가 일정 확정 발송
* 비공개 요청은 경영진만 열람 가능하며, 필요 시 HR을 경유해 후속 조치를 안내합니다.

***

#### 에티켓 & 규칙

1. **사전 준비** – 논의할 주제·자료를 미리 정리(메모, 링크 등)
2. **시간 준수** – 기본 30 분(최대 60 분) / 진행자가 종료 5 분 전 리마인드
3. **상호 존중** – 비판보다는 솔루션 제안, 경청·메모 필수
4. **후속 액션** – 합의 사항·담당자·마감일을 짧게 슬랙에 기록 → 투명하게 공유
5. **비밀 유지** – 비공개 티타임 논의 내용은 상대 동의 없이 외부 공유 금지

> 함께 마시는 30 분의 티타임이 더 나은 조직을 만듭니다.\
> 언제든 편하게 요청해 주세요!
